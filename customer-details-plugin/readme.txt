# Customer Details Plugin

## Description
Manage customer database with CRUD in WordPress. Create WP users automatically when new customers are added. Frontend display via shortcode with search & pagination.

## Installation
1. Upload `customer-details-plugin` folder to `/wp-content/plugins/`
2. Activate the plugin in WordPress admin
3. Use shortcode [cdp_customer_list] on any page to show frontend list

## Notes
- Admin UI supports search & pagination
- AJAX used for frontend and backend
- Activation creates table `wp_cdp_customers`
